# SIMPA - Sistema de Monitoramento de Projetos Acadêmicos

O **SIMPA** é uma plataforma desenvolvida para otimizar o acompanhamento e a gestão de fluxos de projetos dentro do ambiente acadêmico. O sistema surge como uma solução para centralizar informações, prazos e entregas, facilitando a comunicação entre residentes, orientadores e coordenadores.

> Este projeto faz parte da etapa de imersão do **Programa de Residência em TIC 16**, uma parceria entre a **BRISA** e a **UEMA**.

---

## 🚀 Sobre o Projeto
O objetivo principal do SIMPA é transformar a maneira como os projetos acadêmicos são monitorados, substituindo processos manuais por uma interface intuitiva e automatizada.

### Principais Funcionalidades (Previstas)
* **Gestão de Projetos:** Cadastro e acompanhamento de cronogramas.
* **Monitoramento de Prazos:** Alertas e status de entregas.
* **Relatórios Automáticos:** Geração de dados para acompanhamento de desempenho.
* **Interface Responsiva:** Acesso simplificado via web.
